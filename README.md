# SkyRadiusView

-----
## 测试环境：Xcode6，iOS7.0以上
![GIF](1.gif)
## 简介
实现view背景色的四个边角自定义成圆角

-----
## 使用方法
使用方法很简单，只需要在xib或StoryBoard中继承SkyRadiusView，并设置参数即可
详情请参考项目中的示例代码 和 gif图片

## CocosPods
pod 'SkyRaduisView', '~> 1.0.0'
